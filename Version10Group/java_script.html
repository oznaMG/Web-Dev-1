Basic JavaScript

Making Your WebPage Dynamic

Written by
Avery R. Johnson


While you can make cool WebPages just using HTML with CSS, there is a whole other world opened up by introducing Javascript into a WebPage. JavaScript is a scripting language designed for the purpose of working with WebPages. It allows you to change CSS and HTML information. In this lesson we will start learning the basics of intergrating JavaScript into your website, making your webpages more dynamic and interactive.


Copy and Paste this code into your body
<script>
    alert("This is an alert")
</script>

The code you copy and pasted above executes a pop up alert on your browser. You have probably seen this on websites before, usually an annoying alert, telling you an error or problem. Right now, the code executes this as soon as the html reads the scripted code. This means the alert shows up as soon as the page loads, this is as you can guess, an annoying and not so amazing web design feature. We can fix this however, all we need to do is make our own function that will only send an alert upon a certain condition happening. To do this, first remove the script code from the html file so it no longer instnatly reads the alert function. Then, you will need to go into your JS folder and create a file within it named "script.js"
