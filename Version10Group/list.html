How to Create a List in HTML

Step-by-Step Guide

Follow these steps to create an unordered list in HTML:

1. Create a basic HTML structure

Start with the basic structure of an HTML document:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My List</title>
</head>
<body>
    <!-- Your content and list goes here -->
</body>
</html>

2. Add a heading for your list

Include a heading to introduce your list:

<h3>My Favorite Video Games</h3>

3. Create an unordered list

Use the <ul> tag to create an unordered list, and the <li> tags for each list item:

<ul>
    <li>The Witcher 3</li>
    <li>Titanfall 2</li>
    <li>Atomic Heart</li>
</ul>

Full Example

Here is the full code example with all the steps combined:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My List</title>
</head>
<body>
    <h3>My Favorite Video Games</h3>
    <ul>
        <li>The Witcher 3</li>
        <li>Titanfall 2</li>
        <li>Atomic Heart</li>
    </ul>
</body>
</html>

Using CSS to Style Your List

Make a new css file name syle.css

1. Update Your CSS File

/* CSS for styling your lists */
               ul {
               list-style-type: square;
               color: #003366;
                 }
                ol {
                 list-style-type: decimal;
                   color: #003366;
                 }

2. Link Your CSS File in Your HTML

Make sure to link your CSS file between the <head> section of your HTML:

<link rel="style.css" href="style.css">

Inline Styling

Apply inline styling to individual elements for specific styling:

<td style="color:#00FF00; background-color:sienna;">Styled Cell</td>
