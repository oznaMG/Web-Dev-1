Navbar description

by Benjamin | Shakopee High School (SHS) |

A navbar (navigation bar) is a user interface element that typically appears at the top or side of a webpage, providing links or buttons for easy navigation to different sections or pages of a website or application.


Elements of Navbar:

Create Navbar: <nav> Content </nav>

Using this element: "<nav>" and "</nav>" Creates a Navbar which can be used to create a header. Down below is an Example on how to use the Element: 

                                        
                                        
                                        
<nav class="navbar navbar-dark bg-dark"> 
  
                                        
    <!-- Navbar content --!>

                                        
 </nav>


Branding: .navbar-brand
